Health Coaching Management System - Loading Instructions

Since I am working with various databases and a REACT website, in order to load it, certain steps need to be followed. These steps are as follows:

Step 1

Open the product folder in an IDE (preferably Visual Studio Code). Note: the IDE should have a terminal. The IDE screen should look something like this:

Step 2

Now the database will be hosted on a local HTTP server.

Firstly open a new terminal

Then in the terminal, enter the folder where the data is stored. Do so by typing "cd src/data" in the terminal and running the command

The database will be hosted using the json-server package. Ensure that you have the json-server package installed by typing "npm install json-server".
After the installation has completed, host the database by typing the command "npx json-server datastore.json --port 3001"

Step 3

Next, the security and upload server has to be hosted on a local HTTP server as well.

Open a new terminal.
*ENSURE THAT THE EXISTING TERMINAL WITH THE DATABASE HOSTING KEEPS RUNNING*

On this new terminal, enter the AdminSide folder by typing the command "cd src/AdminSide"

Now, run the security/upload server by typing "node server.js" into the terminal

Note: the MAC check has been disabled for the purpose of demonstration, yet its evidence can be found in the source code

Step 4

Finally, run the website

Open a new terminal and type "npm run start". Wait for a few moments till npm builds the website environment. After a while the website should accessible at https://localhost:3000.

In case there is an error in running the website on the system, ensure that node package manager (npm) is updated to the latest version.
